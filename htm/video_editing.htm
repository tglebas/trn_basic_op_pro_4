Video Editing

Video editing skills are essential in order to reduce the voluminous data that can be collected during a typical ROV operation down to the essential requirements necessary to fully document the project in a concise manner. Additional skills beyond basic cutting and cropping can help produce a more compelling presentation.

The following exercises will help develop the core skill set to convert raw video data into professional quality presentations. There are many additional skills within the realm of image and video editing, but these are outside of the scope of this course.

There are many video editing tools available in the marketplace. The VideoRay software configuration includes Microsoft Windows Live Movie Maker and Cyberlink PowerDirector software, which are video editing tools.
